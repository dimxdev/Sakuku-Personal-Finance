# 💰 SakuKu — Personal Finance App

> Kelola keuanganmu dengan mudah, satu genggaman tangan. ✨

SakuKu adalah prototipe aplikasi keuangan pribadi bergaya mobile yang dirancang untuk membantu pelajar dan profesional muda **mencatat pemasukan**, **mengelola pengeluaran**, dan **mencapai target tabungan** — semua dalam satu aplikasi yang bersih dan sederhana. 🌱


---

## 🚀 Fitur Utama

| Fitur | Deskripsi |
|---|---|
| 🏠 **Dashboard Beranda** | Lihat saldo, pemasukan, pengeluaran, dan target tabungan utama sekilas pandang |
| 💸 **Catat Transaksi** | Tambah pemasukan/pengeluaran dengan keypad numerik & kategori bergambar |
| 🎯 **Target Tabungan** | Buat, edit, dan hapus target tabungan sendiri (Create/Update/Delete) |
| 🔔 **Pengingat** | Atur notifikasi pengingat keuangan, bisa diedit & dihapus kapan saja |
| 📊 **Laporan Keuangan** | Visualisasi pemasukan vs pengeluaran & rincian kategori bulanan |
| 🕘 **Riwayat Transaksi** | Cari & filter transaksi berdasarkan tipe, tanggal, dan kategori |
| 💳 **Metode Pembayaran** | Kelola rekening bank & e-wallet — tambah, edit, hapus dengan mudah |
| 🌙 **Mode Gelap** | Satu sentuhan ikon untuk beralih tema terang ⇄ gelap |
| 🌐 **Dwibahasa** | Mendukung Bahasa Indonesia 🇮🇩 dan English 🇬🇧 secara instan |

---

## 🛠️ Tech Stack

- ⚛️ **React 18** + **TypeScript**
- ⚡ **Vite 6** — dev server super cepat
- 🎨 **Tailwind CSS v4** + **shadcn/ui** (Radix UI)
- 🎬 **Motion** — animasi transisi antar layar yang halus
- 📈 **Recharts** — grafik laporan keuangan
- 🔔 **Sonner** — notifikasi toast yang clean

---

## 📦 Menjalankan Proyek

```bash
# 1️⃣ Install dependencies
npm i

# 2️⃣ Jalankan dev server
npm run dev
```

Lalu buka browser ke alamat yang muncul di terminal (biasanya `http://localhost:5173`) dan mulai jelajahi SakuKu! 🎉

---

## 📱 Tentang Tampilan

Aplikasi ini dirender dalam **frame ponsel** di tengah layar — jadi meskipun dibuka di browser desktop, pengalamannya tetap terasa seperti memakai aplikasi mobile asli. 📲
